Write a program that reads a floating-point number representing a temperature in Celsius and converts it to Fahrenheit. The program should then determine if the temperature is above or below the freezing point of water (32°F or 0°C). Print 'Above freezing' if the temperature is above the freezing point, or 'Below freezing' if it is below the freezing point.
Sample Input:
25.5

Sample Output:
Above freezing


Sample Input:
-10

Sample Output:
Below freezing



Memory limit: 256 MB
Time limit: 5 seconds
